Desde la versi√≥n 0.8.0 es posible definir todas las propiedades de los data-cb en el objeto del fragment en lugar de tener que respetar el anidamiento del HTML.

Es decir, esto:

return tpl.fragment({
  box: { style: { borderColor: color } },
  colorText: { inner: color },
  btn: { onclick: changeColor }
});

Debe funcionar igual que esto:

return tpl.fragment({
  box: { 
    style: { borderColor: color }, 
    nodes: {
      colorText: { inner: color },
    } 
  },
  btn: { onclick: changeColor }
});
